# 문의 대응 (Inquiries)

외부/사내에서 들어온 확인·검수 요청을 건별로 정리하는 폴더.

## 규칙
- 파일명: `YYYY-MM-DD-주제-요청자.md`
- 각 문서 상단에 **접수일 / 요청자 / 마감 / 상태 판정** 명시
- 상태 판정은 3택: `포함 O·진행중` / `포함 O·미착수` / `제외`
- 요청 자료(구성도·매트릭스·메시지 예시·설정법·테스트)는 문서 안에 인라인 정리
