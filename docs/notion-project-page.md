# 🎬 make_cc — 프로젝트 워크스페이스

> 한국어 영상을 올리면 **자동 자막(SRT)** 을 만들고, 브라우저에서 **편집**하고, **스타일을 입혀 영상에 박고(번인)**, **영어·일본어·중국어로 번역**까지 하는 풀스택 웹 서비스.

## 🛠️ 기술 스택

| 영역 | 선택 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) · TypeScript |
| 인증/DB/스토리지 | Supabase (Auth · Postgres · Storage, RLS) |
| 큐 | BullMQ + Redis (미연결 시 DB 폴링 워커로 폴백) |
| STT | self-host Whisper large-v3 (ko) + pyannote 화자분리 — GPU 워커 |
| 번인 | ffmpeg + libass (ASS 필터) |
| 번역 | DeepL API (무료 티어) |
| UI | Tailwind · shadcn/ui · Zustand · TanStack Query |
| 알림 | Resend(이메일) · Discord(봇 DM) |
| 테스트 | Vitest (190개 통과) |
